# <img src="app_mark.png" alt="Share My Thing icon" width="36" /> Share My Thing <img src="app_mark.png" alt="Share My Thing icon" width="36" />

A phone, tablet, and wear OS app that allows you to share links, contact details, and more via QR code or plain text, along with widgets, tiles, and complications for your device(s). Syncs between phone & watch if available. No sign-up or accounts required.

## Features

- 3 Item types: Choose between plain text, QR code, or both to influence how the item appears on tiles/widgets/complications and within the app.
- Watch & phone surfaces: Add up to five widgets, five tiles, and five complications, each linked to an item.
- Standalone watch app: Works without the phone app, add or edit items on the watch using the native keyboard. If you want multi-line editing and the ability to reorder items, you can install the mobile app and it will automatically sync.

## Requirements

| | |
|---|---|
| **Phone / tablet app** | Android 11+ (API 30+), Google Play services (for watch sync) |
| **Watch app** | Wear OS 3+ |
| **Sync** | Phone (or tablet) and watch paired; same Google account |
